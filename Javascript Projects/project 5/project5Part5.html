Problem Statement

Begin by examining the program already present in the Part 5 JavaScript library file. The program is complete, except for one of the functions, namely the one that calculates the Body Mass Index (BMI) based on the user's weight and height inputs.

You must complete the program by writing that one function. The formula for the BMI calculation is given in a comment within the JavaScript library file.  No other changes need be made to the program (the set of functions). After the modification, the program must successfully calculate the BMI based on whatever valid weight and height data is entered into the program by the user.

The program must continue to have no inputs other than just the weight and height keyed in by the user (i.e. just as it already does).  Also, the program must continue to work only with local variables. There is no need to set up any global variables for this project part, and it is worth zero (0) points if any global variables are added to the js file.

  • Project Notes:
    • Add in no code to the function without first having decided on a plan for what you will be doing.
    • Proceed with writing the function using 'versions'. For example, version One might be to simply make sure that the flow of control is entering the function. For example, version Two might be to make sure that the two inputs to the function are successfully arriving into the function. And so on. Test each version to make sure it is accomplishing what you planned for it to accomplish, before going ahead with the next version. As is often the case, the  alert()  method will be handy for verifying that a version is working properly.
    • In establishing your test cases, be sure to include one in which the user will have a BMI of less than 25 (that one is already done for you as the sample output), one that will result in a BMI of 25, and one that will result in a BMI that is greater than 25.
