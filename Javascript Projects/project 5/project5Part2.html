Problem Statement

Design and build a program that allows a user to enter a series of names in a text box in a form on a web page. The names are entered one at a time, with the user clicking a submit button to process each name.  (NOTE:  The click of the submit button is what will run your program, and not merely a refresh of the web page itself, as has been the case in most other programs in this course.)

The names are stored (using an array) and as each name is entered the new name is added to a table of names displayed below the form. Displayed above the table there must be a total of how many names have been entered. This total will be calculated from the names in storage and not accumulated. Program modules (functions) must be used for clarity and ease of future maintenance, although no specific number of functions is mandated as long as there are at least two.
