General Project Information

  • The instructor reserves the right to modify the specifications for any of these parts in writing, in an email, or verbally during class time. These changes to the spec will be required.
  • The project consists of four parts. Each part has it's own page and specifications.
  • The course projects are not group projects. You need to do you own work completely.
  • If you need help on a project you should first ask your instructor. Each instructor will have their own procedure for getting help from them.
  • You may talk to classmates but do not share code in any electronic or physical form. Conversations with classmates about projects should be general and not contain details about solutions.
  • Don't sit down at a classmate's computer and type into their project.
  • Don't ask classmates for their projects.
  • Be very careful when you print project files at MATC. When you print go to the printer immediately and retrieve your printouts.
  • There is sample output shown for each part.

Important Note

All projects must be complete and function according to the specifications before they will be graded. There is no partial credit given for incomplete projects, i.e. one with incorrect output. If an incomplete project is submitted you will receive a score of zero (0) in Blackboard, with an accompanying message in Blackboard pointing out which project programs had incorrect output. See the course syllabus for details. In addition to the loss of three (3) Professional Reputation points, late points would also be a factor after the Learning Unit deadline.
