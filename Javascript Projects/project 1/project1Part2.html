Problem Statement

Write a program that gets from the user (i.e. employs the prompt method) the first and last names of a dog owner and the name of their dog. The program will generate an information statement for the dog owner using the following pattern and then displays the string on the page.

  • First Name
  • a space
  • Last Name
  • an apostrophe
  • the letter s
  • a space
  • this string of text: dog is named
  • a comma
  • a space
  • a double quote mark
  • the dog's name
  • a double quote mark
  • a period
